# Micro-interactions

Small, functional UI motion that makes an interface feel responsive and alive: hover/press/focus feedback, toggles, toasts, drawers, and list/layout animation. For frontend developers polishing UI feedback with Framer Motion (`motion/react`) and modern CSS — feedback and continuity, not decoration.

## When it activates

- "Add a hover/press effect" or "animate a toggle or switch"
- "Build an animated like button" or "make a toast/snackbar slide in"
- "Animate a drawer or modal" or "animate list reordering or add/remove"
- "Do a shared-element layout transition" or "polish UI feedback"

## Example prompts

- "Build an animated like button with instant press feedback."
- "Make this toast slide in and out smoothly."
- "Animate list add/remove and reordering."
