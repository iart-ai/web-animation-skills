# Lottie Animation

The bridge from After Effects to product: vector animation shipped as JSON (or zipped `.lottie`), tiny and resolution-independent, played by a runtime on web, iOS, Android, and React Native. For frontend developers integrating, controlling, theming, and exporting Lottie — with dotLottie preferred for new work.

## When it activates

- "Add a Lottie animation" or "play a .lottie or .json animation"
- "Integrate a Bodymovin/After Effects animation on web or mobile"
- "Control Lottie playback / segments" or "make a scroll-driven Lottie"
- "Recolor a Lottie at runtime" or "export an AE animation to Lottie"

## Example prompts

- "Add a scroll-driven Lottie to my landing page."
- "Recolor this Lottie to match my brand at runtime."
- "Control playback segments on this dotLottie animation."
