# Accessible Animation

Honor `prefers-reduced-motion: reduce` as graceful degradation, not a kill switch. For frontend developers building accessible motion — sorting animation into tiers so vestibular-triggering motion is removed while orientation cues and essential feedback stay, in both CSS and JS.

- "Make my animations accessible" or "fix vestibular / motion-sickness issues"
- "Add a useReducedMotion hook" or "gate GSAP / Framer Motion / Lenis for reduced motion"
- "Meet WCAG 2.3.3 / C39"
