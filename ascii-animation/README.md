# ASCII Animation

Render motion entirely with text characters — extremely lightweight, distinctive, and runnable in browsers (`<pre>`/canvas), terminals (ANSI), and over any 3D scene (Three.js `AsciiEffect`). For frontend developers building generative ASCII fields and converting images, video, or 3D scenes into animated ASCII art.

- "Create retro/hacker text-character motion" or "animate text with a brightness ramp"
