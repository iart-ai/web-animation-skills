# SVG Animation

Crisp, lightweight, infinitely scalable vector motion for icons, illustrations, logos, and data marks. For frontend developers animating SVG three ways — CSS, SMIL, and JS (GSAP/Web Animations) — covering stroke draw-on, path morphing, motion-along-path, and animated gradients/filters.

## When it activates

- "Animate an SVG" or "make a line draw itself on"
- "Do a stroke draw-on / signature animation" or "morph one shape into another"
- "Move an element along a path" or "animate an icon/logo"
- "Animate an SVG gradient or filter"

## Example prompts

- "Animate this SVG logo to draw itself on."
- "Morph my menu icon into a close icon."
- "Move this marker along an SVG path."
