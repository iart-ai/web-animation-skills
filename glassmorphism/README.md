# Glassmorphism

Frosted, translucent UI that reads as real glass: backdrop blur, a bright edge highlight, layered shadow, and the 2026 "liquid glass" look — refracting, slowly churning surfaces with a specular sweep. For frontend developers building glass cards, nav bars, modals, and heroes that stay legible and accessible.
